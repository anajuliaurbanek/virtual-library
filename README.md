# 📚 Virtual Library

Uma biblioteca virtual inspirada em estantes clássicas, desenvolvida para organização pessoal de livros.

## ✨ Objetivo

Criar uma experiência visual de biblioteca digital onde os livros possam ser organizados livremente em prateleiras.

## 🚀 Tecnologias

- React
- JavaScript
- PHP
- MySQL
- CSS

## 📋 Funcionalidades Planejadas

### V1
- [ ] Cadastro de livros
- [ ] Edição de livros
- [ ] Exclusão de livros
- [ ] Upload de capa
- [ ] Upload de lombada
- [ ] Organização em estantes

### V2
- [ ] Drag and Drop
- [ ] Avaliações
- [ ] Metas de leitura
- [ ] Tags
- [ ] Busca

## 🎨 Inspiração

Bibliotecas clássicas em tons de azul e branco com visual elegante e personalizável.
